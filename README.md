# Fdev
Flutter development environment

### For Linux
* Downloads Flutter SDK
* Add it to PATH
* Install Android Studio

##### Usage:

make the finstall.sh file runnable,

```
chmod -x finstall.sh 
```

then run the file .sh to installs

```
./finstall.sh  
```
 
### For Windows
* Install chocolatey
* Install Git
* Download Flutter SDK
 
##### Usage:

`run the .bat file as Admin`
